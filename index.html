Seth's Cheat Sheet

Tag Name

Formatted Tag

Tag Description
Header <header> Describes content at the top of the page <body>
Nav <nav> Encapsulates the page's navigational links. It is often placed inside the <header> or <footer>
Main <main> The main content of the page between the header/navigation and the footer areas
Footer <footer> Content at the bottom of the <body>
Video <video> Allows us to add videos to our website. Requires a closing tag.
Audio <audio> Allows us to put audio on our website. Requires a closing tag.
Embed <embed> Can be used to implement any type of media.
Figure <figure> This element is used for encapsulating media such as an image, a diagram, or a snippet of code.
Figcaption <figcaption> This element is used to describe the media within the <figure> element
Section <section> Defines elemnets in a document, like chapters, headings, or any other area of the document within the same theme.
Article <article> This element is used to hold content that makes sense with the theme.
Aside <aside> This element is used to mark additional info that can enhance another element but isnt required to understand the main content.
